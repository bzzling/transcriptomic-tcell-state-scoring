# Data Availability Statement

This project uses the publicly available processed RNA-seq data from GSE122149 associated with:

Ogando J, Sáez ME, Santos J, Nuevo-Tapioles C, Gut M, Esteve-Codina A, Heath S, González-Pérez A, Cuezva JM, Lacalle RA, Mañes S. PD-1 signaling affects cristae morphology and leads to mitochondrial dysfunction in human CD8+ T lymphocytes. J Immunother Cancer. 2019 Jun 13;7(1):151. doi: 10.1186/s40425-019-0628-7. PMID: 31196176; PMCID: PMC6567413.

The downloaded GEO data files are not tracked in this repository.

## Quick Start

1. Visit https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE122149.
2. Download the series matrix file and the normalized counts supplementary file.
3. Copy them into the `data/raw/` directory as is without extracting
4. Verify that there are two files, namely `GSE122149_series_matrix.txt.gz`, and `GSE122149_Smanes_normalized_counts.txt.gz`.
